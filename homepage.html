The Geometry Dash Personal Demonlist

Now on a new Website as opposed to the old bad one (it was really bad)Google Site!

Created by Li MaoTM

This Website was created as a way to rank completed Demons that were completed from a certain group of people.

Record Submission Rules

  • Achieved the record without using hacks (however, FPS bypass is allowed, up to 360fps).
  • Achieved the record on the level that is listed on the site - please check the level ID before you submit a record or submit a new level.
  • We accept new levels but only those of Hard Demon Difficulty or above as to not spam the List with random Levels. This Rule is highly subject to change to higher requirement, for example Insane Demons and above.
  • 2 Player Demons need to have been beaten SOLO to be accepted.
  • Do not use secret routes or bug routes, you can use skips as long as they don't skip a whole part and instead just skip for example 2-3 clicks.
  • Do not use easy modes, only a record of the unmodified level qualifies.
  • Please do not submit multiple records of the same record (don't submit dupes or i will smite you, or just not accept the record, have patience)!
  • Have either source audio or clicks/taps in the video. Edited audio only does not count (Exceptions may be applicable)
  • Once a level falls onto the Legacy List, we do not accept any more records for it, the only exception to this is if you beat it while it still was on the List but never submitted it or we missed it.
  • If you aren't sure if something is allowed or not, you can refer back to the Pointercrate Guidelines: https://pointercrate.com/guidelines/index
